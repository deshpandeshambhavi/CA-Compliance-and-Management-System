# 📋 Client & Compliance Management System

A web-based Client and Compliance Management System designed for Chartered Accountants (CAs), accounting firms, and financial service providers to efficiently manage clients, services, billing, and regulatory compliance.

Built using **Python, Flask, MySQL, HTML, CSS, and Bootstrap**, the system provides a centralized platform for tracking client information, assigned services, compliance deadlines, and billing status.

---

## 🚀 Features

### 👥 Client Management
- Add new clients.
- Store client details:
  - Name
  - PAN Number
  - Phone Number
  - Email Address
- View complete client profiles.
- Generate client-specific reports.

---

### 💼 Service Management
- Add accounting or consultancy services.
- Define service charges.
- Assign services to clients.
- Track service usage history.

---

### 💰 Billing Management
- Generate bills for clients.
- View billing history.
- Track payment status.
- Mark bills as paid.
- Automatically update overdue bills using stored procedures.

---

### 📑 Compliance Management
- Add compliance records.
- Associate compliances with clients.
- Link compliances to regulations.
- Track:
  - Compliance type
  - Due date
  - Current status
- Update compliance status.

---

### 📊 Dashboard
Displays important business metrics:

- Total clients
- Total services
- Unpaid bills
- Overdue compliances

Also shows upcoming compliance deadlines.

---

### 📈 Reports
Generate useful business insights:

- Revenue generated by each service.
- Clients using multiple services.
- Overdue compliance reports.
- Individual client reports.

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/client-compliance-management.git

cd client-compliance-management
```

---

### 2. Create a virtual environment

Windows:

```bash
python -m venv venv

venv\Scripts\activate
```

Linux/Mac:

```bash
python3 -m venv venv

source venv/bin/activate
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Configure MySQL

Create a MySQL database:

```sql
CREATE DATABASE ca_management;
```

Import the required tables and stored procedures.

Update database credentials inside:

```
app.py
```

```python
DB_CONFIG = {
    'host': 'localhost',
    'user': 'root',
    'password': 'your_password',
    'database': 'ca_management'
}
```

---

### 5. Run the application

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000/
```
